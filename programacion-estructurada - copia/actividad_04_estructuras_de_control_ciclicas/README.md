# Actividad: Estructura de control cíclicas
## Descripción
Programas que ejecutan estructura de control cíclicas (while, for, do-while) simulando menús y repasando las estructuras de cada ciclo.
## Temas aplicados
- Operadores.
- Estructuras de control cíclicas.
## Compilación
gcc do_while.c for.c while.c -o estructuras_de_control_ciclicas
## Ejecución
./estructuras_de_control_ciclicas
