# Actividad: Matrices.
## Descripción
Programa que simula el funcionamiento del juego "BattleShip" usando como núcleo las matrices bidimensionales.
## Temas aplicados
- Operadores.
- Arreglos.
- Estructuras de control condicionales.
- Estructuras de control cíclicas.
- Librería stdlib.h
- Matrices.
## Compilación
gcc matrices_coordenadas.c matrices.c -o matrices
## Ejecución
./matrices
