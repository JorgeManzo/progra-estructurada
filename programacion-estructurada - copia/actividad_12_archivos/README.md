# Actividad: Archivos.
## Descripción
Programa que demuestra  la lectura, escritura y manipulación de archivos externos utilizando la libreria stdio.h del lenguaje C.
## Temas aplicados
- Operadores.
- Procedimientos.
- Punteros.
- Direcciones de memoria.
- Archivos de texto.
- Matrices.
- Estructuras de control cíclicas.
## Compilación
gcc matriz_archivos.c tablero_archivos.c -o archivos
## Ejecución
./archivos
