# Actividad: Arreglos
## Descripción
Esta actividad hace refrencia al uso de arreglos para representar nombres completos, listas de nombres y como recorrer estas listas.
## Temas aplicados
- Operadores.
- Arreglos.
- Estructuras de control cíclicas.
- Librería stdlib.h
## Compilación
gcc arreglos_direccion_mem.c arreglos_nombres.c arreglos_temperatura.c -o arreglos
## Ejecución
./arreglos
