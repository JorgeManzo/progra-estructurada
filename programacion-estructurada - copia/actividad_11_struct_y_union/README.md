# Actividad: Struct y union.
## Descripción
Programa que muestra la creación de tipos de datos compuestos personalizados. Se utiliza struct para agrupar múltiples variables bajo un mismo nombre y union para optimizar recursos compartiendo un mismo espacio de memoria entre diferentes variables.
## Temas aplicados
- Operadores.
- Estructuras de control cíclicas.
- Arreglos.
- Estructuras de datos.
## Compilación
gcc recorrido_struct.c struct.c union.c -o structs_y_unions
## Ejecución
./structs_y_unions
