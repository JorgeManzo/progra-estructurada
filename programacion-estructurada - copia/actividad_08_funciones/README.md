# Actividad: Funciones.
## Descripción
Uso de funciones para modularizar progrmas que van desde sumar enteros hasta recorrer arreglos, tambien se compararon los tipos de paso de variables a funciones.
## Temas aplicados
- Operadores.
- Arreglos.
- Estructuras de control condicionales.
- Estructuras de control cíclicas.
- Funciones.
- Punteros.
## Compilación
gcc funcion_de_paso.c funcion_sumar.c funciones_arreglos.c funciones_matriz.c funciones_paso_parametro.c -o funciones
## Ejecución
./funciones
