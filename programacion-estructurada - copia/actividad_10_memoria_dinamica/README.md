# Actividad: Memoria dinámica.
## Descripción
Programa que reserva y libera memoria en tiempo de ejecución (usando malloc y free) que se guardan en un arreglo unidimensional.
## Temas aplicados
- Operadores.
- Punteros.
- Direcciones de memoria.
- Estructuras de control cíclicas.
- malloc.
- Arreglos.
## Compilación
gcc malloc.c -o malloc
## Ejecución
./malloc
